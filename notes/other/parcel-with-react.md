- 패키지 매니저: yarn
- 번들러: Parcel



### yarn, git init

```sh
$ git init
$ yarn init
```



### Parcel을 devDependencies로 설치

```sh
$ yarn -D add parcel-bundler
```



### react, react-dom 설치

```sh
$ yarn add react react-dom
```



### babel 설치

```sh
$ yarn -D add @babel/core @babel/preset-env @babel/preset-react
```
